# アンガーダイアリー
アンガーダイアリーとはイラッとした内容を記録することで、自分の怒りの傾向を知り、感情を制御できるようになり、精神を安定させる助けになるアプリです。

# サイトリンク
https://anger-diary.herokuapp.com

# 使用技術
* Ruby 2.6.5
* Ruby on Rails 6.0.3.4
* PostgreSQL 12.2
* JavaScript
* Bluma
* Slim
* Heroku
* GitHub

# インストール
```
$ bin/setup
$ rails server
```

# テスト
```
$ bin/rspec
```

# Lint
```
$ rubocop -a
```
